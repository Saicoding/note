#学习文档
* [官方文档](https://angular.cn/)
* 在模版里使用属性
    ```
      <option *ngFor="let pow of powers" [value]="pow">{{pow}}</option>
    ```
* 当在表单中使用[(ngModel)]时，必须要定义name属性。

* 在模版中调用{{diagnostic}}可以只能得到值
    ```javascript
          get diagnostic() {
             return JSON.stringify(this.model);
          }
    ```
