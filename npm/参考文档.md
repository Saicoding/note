# 常用指令

* 全局路径查询

    ```
        npm root -g
    ```

* 查看安装过的包

    ```
        npm list -g --depth 0
    ```
* 查看npm配置

    ```
        npm config list //查看基本配置  npm config list -l //查看所有配置
    ```    

* 怎么完全卸载ionic和cordova并重新安装
    ```
        npm uninstall cordova ionic
    ```

* 安装最新版本的npm
    ```
       npm install --save latest-version
    ```
