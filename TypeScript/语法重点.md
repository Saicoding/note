# 语法重点

* 定义数组
```javascript
    let list: number[] = [1, 2, 3];
    或
    let list: Array<number> = [1, 2, 3];
```

* 解构数组妙用
    * 数值替换
    ```javascript
        let [first,second] = [second,first];
    ```
    * 作用于函数参数
    ```javascript
        let input=[1,2]; 
        function test([first,second]:[number,number]){
            console.log(first);
            console.log(second);
        }

        test(input)
    ```
    * 属性重命名
    ```javascript
        let o = {a: "foo",b: 12,c: "bar"};
        let { a: newName1, b: newName2 } = o;
        相当于：
        let newName1 = o.a;
        let newName2 = o.b;
    ```
