# 常用指令

* [官方参考文档](https://ionicframework.com/docs/cli/)

* 创建新app
    ```
        ionic start myNewProject
    ```

* 安装ionic
    ```
        npm install -g ionic@latest
    ```

* 添加服务
    ```
        ionic g provider MyData 
    ```
* 添加页面
    ```
        ionic g page myPage 
    ```

* 查看是否连接设备
    ``` 
         adb devices
    ```
