# 小知识点

* 手机应用图片一般分辨率为72像素/英寸 海报打印是300像素/英寸

* Ctrl + t 自由变换

* ctrl + j 或者alt+拖动图片 复制图层 

* alt + delete 将选区填充为前景色

* ctrl + d 取消选区

* alt + 鼠标右键 +拖动鼠标

* 颜色替换工具是替换颜色的，点击连续就是颜色差别大的不会有效果

* 历史画笔记录工具

* 扣发丝可以用背景橡皮擦工具

* 加深和减淡一般用于人像

* ctrl+ enter 将路径变选区

* 建立好路径后，可以选择描边路径，可以做出圆滑的线条

* ctrl +shift +x 调出液化

* ctrl + alt + shift + E 合并图层
